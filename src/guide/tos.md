# SuzuBot 服务协议

:::tip
更新日期：2023-01-19
:::

本页中的服务协议阐述了您与 SuzuBot 之间关于使用 SuzuBot 以及其服务的关系。

邀请 SuzuBot 进入群组、添加 SuzuBot 好友或主动使用 SuzuBot 的任何功能均视为您已经阅读并同意以下条款，如果您不同意以下条款，您就无法使用 SuzuBot 提供的所有服务。

您可以在任何时候通过访问本页来阅读这些条款，我们保留改变、修改、增加或删除本协议的部分的权利，并将修改后的协议同步更新在本页。如果您在这之后仍然使用 SuzuBot 的服务，即表明您同样接受更新后的协议。

## 1. 术语解释

为了使本协议更加清晰简洁，本协议使用简短的术语来表达某些概念。

* SuzuBot：SuzuBotDev 开发的 SuzuBot IM 机器人程序以及其在各个 IM 下的载体账号。
* 服务：SuzuBot 上提供的一切模块、功能。
* 协议：本协议中所有条款的统称。
* 风控：IM 账号被平台认定为风险账号并使其无法发送或接收消息、无法添加好友或群组、暂时或永久停止登入等等影响服务的限制。
* 我们：SuzuBot 的开发者，SuzuBotDev 项目的所有贡献者。
* 用户：SuzuBot 的使用者，也就是**您**。

## 2. 许可和使用限制

### 2.1 在使用 SuzuBot 的服务前，您应当知晓 SuzuBot 服务的各项使用方法以及正确命令格式。您应当知晓，任何人没有任何义务来指导您使用 SuzuBot 的服务。

### 2.2 使用 SuzuBot 的服务需要您具有理解自身行为的基本能力，并且知晓和承担使用服务带来的结果。

### 2.3 将 SuzuBot 拉入群组前，请确认您是该群的管理员或拥有者，或该邀请行为已得到多数管理员的同意。

### 2.4 您同意不得展开和从事以下行为

1. 任何可能会提高或导致 SuzuBot 被风控的行为，包括但不限于滥用 SuzuBot 的服务、直接踢出 SuzuBot、禁言 SuzuBot、举报 SuzuBot。
2. 多次将 SuzuBot 拉入人数小于 25 人的群组。
3. 以任何形式宣称 SuzuBot 的所有权。
4. 利用 SuzuBot 的程序漏洞来行使不正当行为。
5. 干扰、破坏 SuzuBot 的服务或 SuzuBot 的服务的服务器和网络。
6. 未经允许的情况下以任何形式将 SuzuBot 的服务转接到第三方。
7. 使用 SuzuBot 骚扰他人或干扰其他 Bot 的正常服务。

### 2.5 我们保留在这些条款下未明确授予您的任何权利。

## 3. 隐私

### 3.1 使用 SuzuBot 的服务将会收集和记录有关您使用 SuzuBot 服务的信息，包括但不限于使用次数，使用场合，使用时间，部分需要绑定的功能保存的数据。

### 3.2 SuzuBot 的部分服务可能使用了第三方 API，这些服务保存的数据将会以业界标准的加密方式进行通信传输以用于为您提供服务。

## 4. 服务的终止

### 4.1 若发现您从事第2节中的任何行为，我们有权利终止对您进行限制。

限制的方式包括但不限于:
1. 拒绝对您提供任何服务
2. 拒绝您发出的一切好友请求和群组邀请
3. 拒绝对您所担任管理员的群组提供任何服务

### 4.2 若您组织或者参与了对 SuzuBot 以及其开发人员无故进行骚扰、威胁、污蔑、诋毁等恶意行为，我们有权利终止对您进行限制。

### 4.3 若发现您从事没有在这些条款中列出，但可能会影响他人自由使用 SuzuBot 服务的行为，我们有权利终止对您进行限制。

### 4.4 对于首次违反协议的行为，我们会进行口头警告。对于多次违反协议的行为，我们将会加大限制的力度。

## 5. 责任和约束

### 5.1 我们不对您在使用 SuzuBot 的服务的过程中可能发生的任何损失负责，不论这些损失是何种形式的。

### 5.2 SuzuBot 的部分服务使用的第三方 API 可能会违反您与第三方的服务条款，您将自行承担使用这些服务带来的一切损失。

### 6.1 如果您对 SuzuBot 以及其服务有任何建议和反馈，请通过 SuzuBot 认领群来进行反馈，或发送邮件至 photonspk@gmail.com，您需要清晰的表达您的需求和意见以方便我们改善 SuzuBot 以及其服务。
