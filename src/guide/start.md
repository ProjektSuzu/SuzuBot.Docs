# 开始使用

如需使用 SuzuBot ，则至少需要添加好友或者和 SuzuBot 处在同一群聊内。

## 添加好友

如果您希望单独使用 SuzuBot 的功能而不希望在群组中使用，可以选择添加 SuzuBot 为好友。

可通过以下任意一种方式申请添加 SuzuBot 的好友（按照推荐度排序）：
- 通过加入 SuzuBot 官方 QQ 群（955578812）进行申请
- 通过 QQ 号直接搜索申请（2706409070）

需要注意的是，好友申请需要回答问题。请清楚直接的阐明来源和目的，模糊不清的答案可能不会通过。

::: tip 正确的申请
从何处得知Bot:

  `从 XXX 群来 / 群友推荐 / SuzuBot 网站`

想要使用哪些功能:

  `使用 XXX 功能`
:::

::: danger 错误的申请
从何处得知Bot:

  `我不到啊`

想要使用哪些功能:

  `小亮给他整个活`
:::

## 邀请群聊

如果希望 SuzuBot 为您所在的群组提供服务，则需要邀请 SuzuBot 加入您的群组。

在此之前，请确认您符合以下条件：
- 群组成员大于或等于 25 。如该群是正在发展的新兴群聊，则本条限制可放松
- 群组管理员同意 SuzuBot 加入群组
- 群组内同类型机器人数量不超过 1 个
- 群组管理员未被纳入拒绝服务名单

向 SuzuBot 发送群组邀请需要事先添加好友，具体操作请参阅 [添加好友](#添加好友)

::: warning
您应当发送 邀请入群 而非 推荐群聊
:::

群组邀请发送后，一般会立即自动通过邀请。如果没有及时通过，请与开发者联系。
