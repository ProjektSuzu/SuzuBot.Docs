# 常见问题

# Bot 怎么没反应了

如果在群聊中 Bot 突然不回复了，大概率是被腾讯风控了（

请耐心等待开发者解除风控，或者私聊 Bot 使用

当然，如果连私聊都无法使用，说明 Bot 被暂时封禁，请耐心等待开发者解禁（

# 使用 Bot 时出现出错文本

正常，又出bug了（

如果偶尔出现几次不必担心，多次出现请联系开发者

# XXX 功能怎么用不了

请先检查这个功能是否在当前群聊启用

如果确实启用了，那么请将这个情况报告给开发者，大概率是又出bug了（
