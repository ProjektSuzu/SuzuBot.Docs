# 卫星观测 <Badge type="danger" text="开发中" vertical="middle"/>

提供观测空间飞行器的轨道等功能

数据来源: [celestrak.org](https://celestrak.org/) [heavens-above.com](https://www.heavens-above.com/)

## 卫星轨道

提供目标卫星轨道相关数据和卫星视角俯视图

用法:
```
/orbit <空间飞行器目录编号>
/orbit <空间飞行器目录名称>
/orbit <国际卫星标识符>
```

## 卫星信息

提供目标卫星记录信息

用法:
```
/satellite <空间飞行器目录编号>
/satellite <空间飞行器目录名称>
/satellite <国际卫星标识符>
```
